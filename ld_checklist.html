Quick Reference

Cerner shortcuts, order sets, and how-tos

🖨 MSI Printer Shortcuts

  • Print armband: 1 "%"
  • Print facesheet: 11 "$"
  • Print patient stickers: pre-reg tab "enter"

🧪 Specimen Labs Process

  • Handheld printer at the nurse station
  • Click "Specimen collection" in Cerner toolbar
  • Checkmark the lab you are drawing
  • Scan patient wristband
  • Print
  • Place label on drawn specimen and write initials, date, time

🔗 Connecting FetaLink to Patient Bed

  • Click the box to the left of room number to open up that strip
  • Click "associate"
  • Scan patient wristband
  • Backtime the start time to when the patient came into the room

🧬 Placenta to Pathology

  • Label on the side and top of the white container
  • Order: placenta tissue
  • Print order: right click → select print
  • Facesheet and order inside biohazard bag (clear bag w/ red rim) with placenta inside
  • Walk placenta to pathology

📂 Placing Orders — Path

  • MCHS → Inpatient summary → order entry / new order → shared → Langlais, Beth → green button

🌸 Order Sets — Triage

  • Triage: Langlais, Beth → Favorites → FHMC Triage / Outpatient orders

🤰 Order Sets — Maternal Admit

  • Labor admit: search for provider favorites
  • C/S admit: search for provider favorites
  • Admit to Obs: n/a → keep as triage for up to 24hrs
  • Antepartum admission: search for provider favorites

🩸 Order Sets — Diabetes / Hypoglycemia

  • Maternal hypoglycemia: Langlais, Beth → Favorites → FHMC Diabetes: OB hypoglycemia treatment
  • Antepartum / Postpartum SQ insulin: Langlais, Beth → Favorites → FHMC Diabetes: antepartum/postpartum SQ insulin
  • Labor SQ insulin: Langlais, Beth → Favorites → FHMC diabetes: Labor SQ insulin
  • Labor IV insulin: Langlais, Beth → Favorites → FHMC diabetes: OB IV insulin

⚠️ Order Sets — Preeclampsia

  • Preeclampsia / GHTN: add order → OB preeclampsia and other hypertensive disorders of pregnancy
  • Additional preeclampsia BP meds: suborder → OB preeclampsia anti-hypertensive acute control
  • Eclampsia: add order → OB eclamptic seizure orders

🆘 Order Sets — Maternal Hemorrhage

  • MOH: add order → OB hemorrhage multiphase powerplan
  • MOH PP recovery: Langlais, Beth → Favorites → FHMC OB modified postpartum care: stage 2 (3)

🦠 Order Sets — Maternal Infection

  • Chorio antibiotics / labs: add order → OB chorioamnionitis / endometritis
  • Maternal sepsis: Langlais, Beth → Favorites → Sepsis maternal orderset
  • Severe maternal sepsis: Langlais, Beth → Favorites → Severe Sepsis maternal orderset

💉 Order Sets — Maternal Misc

  • Amnioinfusion: add order → OB amnioinfusion
  • Increase Pit to 30: Langlais, Beth → Favorites → FHMC OB increased upper limit oxytocin orders
  • Delivery medication, delivered PTA: Langlais, Beth → Favorites → FHMC post delivery medications
  • Epidural: Tsau, Kang → 1 cama OB epidural (level) most recent

👶 Order Sets — Newborn

  • Newborn admit: Langlais, Beth → Favorites → FHMC newborn: routine care
  • Late preterm newborn (35.1-36.6w): Langlais, Beth → Favorites → FHMC newborn: late preterm pathway
  • Newborn Hypoglycemia: Langlais, Beth → Favorites → FHMC newborn hypoglycemia glucose gel treatment
